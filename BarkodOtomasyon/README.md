# Barkod Otomasyon

## Overview
Barkod Otomasyon is a barcode management system designed to facilitate the creation, storage, and retrieval of barcode information associated with products. This application provides a user-friendly interface for managing barcodes and products efficiently.

## Project Structure
The project is organized into several directories, each serving a specific purpose:

- **src/Forms**: Contains the main form and its designer file for the user interface.
- **src/Data**: Includes the database context and connection string management.
- **src/Models**: Defines the data models for barcodes and products.
- **src/Services**: Contains service classes for handling business logic related to barcodes and products.
- **src**: The main entry point of the application.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the project in your preferred C# development environment.
3. Restore the project dependencies.
4. Update the connection string in `src/Data/ConnectionString.cs` to match your database configuration.
5. Build the project.
6. Run the application.

## Usage Guidelines
- Use the main form to add, edit, or delete products and their associated barcodes.
- Ensure that the database is properly configured and accessible before running the application.
- Follow the prompts in the application for user interactions.
